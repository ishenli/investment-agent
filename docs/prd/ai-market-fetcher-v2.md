## 市场信息抓取工具 V2

市场信息抓取工具 V2 是对原有市场信息抓取工具的升级版本，采用分步操作流程，提供更好的用户体验。

### 功能概述

#### 1. 分步操作流程
新版本采用三步操作流程：
1. **获取内容** - 选择输入方式（URL抓取或手动录入）并获取内容
2. **AI分析** - 让AI对内容进行总结和归纳
3. **保存数据** - 确认并保存到数据库

#### 2. 支持多种输入方式
- **URL抓取** - 输入网页地址自动抓取内容
- **手动录入** - 手动输入市场信息内容

#### 3. AI智能分析
- 自动生成内容标题
- 提取关键信息点
- 分析内容情感倾向
- 生成内容摘要
- 评估内容重要性

### 用户体验改进

#### 1. 更直观的操作流程
通过步骤指示器，用户可以清楚地知道当前处于哪个步骤，以及还有哪些步骤需要完成。

#### 2. 实时反馈
每个步骤都有明确的成功或失败反馈，帮助用户了解操作状态。

#### 3. 灵活的输入方式
用户可以选择最适合的输入方式，无论是从网页抓取还是手动录入。

### 技术实现

#### 1. 组件架构
- 使用 Stepper 组件实现分步操作流程
- 每个步骤都有独立的状态管理
- 支持步骤间的导航和状态保持

#### 2. 状态管理
- 使用 React useState 进行状态管理
- 每个步骤都有独立的状态变量
- 支持错误处理和加载状态显示

#### 3. API 集成
- 与现有的市场信息抓取 API 保持兼容
- 支持 URL 抓取和手动录入两种方式
- 集成 AI 分析功能

### 使用说明

#### 1. 访问页面
用户可以通过侧边栏导航访问"市场信息抓取V2"页面。

#### 2. 选择输入方式
在第一步中，用户需要选择输入方式：
- **URL抓取** - 点击"通过URL抓取"卡片，输入网页地址进行抓取
- **手动录入** - 点击"手动录入"卡片，手动输入内容

#### 3. 获取内容
根据选择的输入方式，完成相应的内容获取操作。

#### 4. AI分析
系统会自动进行AI分析，用户也可以点击"开始AI分析"按钮手动触发。

#### 5. 保存数据
确认分析结果无误后，点击"确认保存"按钮将数据保存到数据库。
